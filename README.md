# 🏛️ Каталог Музея (Museum Catalog System)

## Описание выполненной работы

Данный проект представляет собой клиентскую часть информационной системы для управления каталогом музея. Приложение разработано на платформе WPF (Windows Presentation Foundation) и позволяет просматривать экспонаты, управлять пользователями и обеспечивать безопасный доступ к данным.

В рамках выполнения практических работ были реализованы следующие функции:

1.  **Авторизация и Регистрация (ПР14):**
    *   Реализован интерфейс входа в систему с проверкой учетных данных.
    *   Создана форма регистрации новых пользователей с валидацией полей (email, совпадение паролей).
    *   Разделены права доступа (Роли: Администратор, Менеджер, Пользователь).
    *   Используется фирменный стиль в темной цветовой гамме (IDE-style).

2.  **Отображение данных (ПР16):**
    *   Разработана страница **CatalogPage** для просмотра списка экспонатов.
    *   Данные отображаются в элементе `ListView` с использованием `WrapPanel`, формирующим сетку карточек.
    *   Каждая карточка содержит изображение (из BLOB-поля БД или заглушку), название, описание и визуальные индикаторы статуса/состояния.
    *   Реализован поиск и фильтрация данных.

## 🛠 Технологии

*   **Язык программирования:** C#
*   **Платформа:** .NET Framework / .NET (WPF)
*   **Разметка интерфейса:** XAML

##  Запуск проекта

1.  Убедитесь, что у вас установлен **Visual Studio** и **SQL Server**.
2.  Откройте решение (`pr13-14.sln`) в Visual Studio.
3.  Настройте строку подключения к базе данных в файле `AppContainer.cs` или в коде страниц (в переменной `ConnectionString`).
4.  Скомпилируйте проект: **Ctrl + Shift + B**.
5.  Запустите приложение: **F5**.
